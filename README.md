# La Fogata 🔥

Cuartel General de la Hermandad — Aplicación web privada para el grupo D&D.

## Estructura

```
index.html   → La Fogata (hub principal, login, secciones, acceso al Dungeon)
dungeon.html → Dungeon (juego roguelike, se abre en pestaña nueva)
```

Un solo link para compartir: `index.html`. El Dungeon se abre desde ahí (botón ☠️ en el hub y en la barra superior), en una pestaña nueva, sin necesidad de un segundo link.

## Acceso

- **Hermandad**: contraseña `fogata`
- **Dungeon Master**: usuario `Nicolás` + contraseña `admin`

## Juego: Dungeon

Roguelike inspirado en Rogue (1980).
- 10 pisos · 15 tipos de enemigos · 3 clases
- Sistema de hambre · Equipo real · PERMADEATH
- Guardar partida automático en las escaleras
